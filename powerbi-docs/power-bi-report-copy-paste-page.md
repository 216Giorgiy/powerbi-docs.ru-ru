# <a name="duplicate-a-report-page-in-power-bi"></a>Дублирование страницы отчета в Power BI
1. В Power BI откройте отчет в [режиме редактирования](service-reading-view-and-editing-view.md) и [создайте визуализацию](power-bi-report-add-visualizations-i.md), если в отчете еще нет этого элемента. 
2. На верхней панели управления выберите значок **Дублировать эту страницу** .
   
   ![](media/power-bi-report-copy-paste-page/pbi_duplicate_new.png)
   
   Будет создана новая страница, которая станет активной.
3. При необходимости [переименуйте страницу отчета](service-rename.md).
